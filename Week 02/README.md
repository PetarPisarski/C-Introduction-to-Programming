## Task 4 (Arithmetic Operations, Relational Operations, Logical Operations)

Create a program that reads two integers from the standard input (`x` and `y`) and prints the result of the following operations to the standard output.

### Arithmetic operations

- `x + y` (addition)
- `x - y` (subtraction)
- `x * y` (multiplication)
- `x / y` (integer division)
- `x % y` (modulo operation)

### Relational operations

- `x < y` (less than)
- `x > y` (greater than)
- `x <= y` (less than or equal)
- `x >= y` (greater than or equal)
- `x == y` (equal)
- `x != y` (not equal)

### Logical operations

- `x && y` (logical AND)
- `x || y` (logical OR)

$$

$$
